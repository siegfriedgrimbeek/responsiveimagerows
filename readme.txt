Allows you to insert a row of images!

== Description ==

Really simple, a user adds a new row and selects the amount of images which should be in the row. The plugin will allow users to add up to five images.

The images will fall underneath each other on mobile.

Upon installation, the plugin will be added to the text editor toolbar.

== Installation ==

Installing "Responsive Image Rows" can be done either by searching for "WordPress Plugin Template" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
